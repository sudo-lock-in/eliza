# Eliza
Runs in the console from the eliza folder.
```
java ProjectEliza.jaza
```
## Functionality

### Starting and Ending
<ul>
  <li>On startup, Eliza will greet you and ask your name.</li> 
<li>After telling her your name, you can quit by typing "exit".</li>
 <li>When you say "exit", she will ask if you would like to start another session. Say "no" to terminate or "yes" to start again.</li>
</ul>

### Responses to Input
<ul>
  <li>
    Responds to exclamations with "Wow! Dramatic! [randomized statement w/ word from input]"
  </li>
</li>
  <li>
  Responds to regular sentences with a randomized statement with a word from input. 
</li>
  <li>
    Responds to questions with a randomized question with a word from input.
  </li>
</ul>
